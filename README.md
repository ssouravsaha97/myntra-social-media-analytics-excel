# 📊 Myntra Social Media Analytics (Excel Project)

## 📌 Project Overview
This project analyzes **Myntra's social media marketing performance** using **Microsoft Excel**.  
The goal is to understand engagement trends, campaign effectiveness, and follower retention across different platforms.

The analysis focuses on identifying key insights that can help improve **content strategy, marketing campaigns, and audience retention**.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Analysis
- Data Visualization

---

## 📂 Project Structure

```
myntra-social-media-analytics-excel
│
├── excel_files
│   ├── myntra_social_media_data_cleaning.xlsx
│   ├── myntra_social_media_engagement_analysis.xlsx
│   ├── myntra_platform_performance_analysis.xlsx
│   ├── myntra_hashtag_content_strategy.xlsx
│   ├── myntra_campaign_effectiveness_analysis.xlsx
│   ├── myntra_follower_retention_analysis.xlsx
│   └── myntra_social_media_dashboard.xlsx
│
└── README.md
```

---

## 📊 Analysis Performed

### 1️⃣ Data Cleaning
- Removed duplicates and inconsistent records
- Standardized column formats
- Prepared dataset for analysis

### 2️⃣ Engagement Analysis
- Measured likes, comments, and shares
- Identified top performing posts
- Analyzed engagement trends

### 3️⃣ Platform Performance Analysis
- Compared engagement across social media platforms
- Identified the most effective channels

### 4️⃣ Hashtag & Content Strategy
- Analyzed hashtag usage
- Determined which content types perform best

### 5️⃣ Campaign Effectiveness
- Evaluated marketing campaign performance
- Compared engagement across campaigns

### 6️⃣ Follower Retention Analysis
- Identified audience retention patterns
- Measured follower engagement trends

### 7️⃣ Excel Dashboard
- Created an interactive dashboard
- Visualized key performance metrics

---

## 📈 Key Insights
- Engagement varies significantly across social media platforms.
- Strategic hashtag usage improves content reach.
- Certain campaigns outperform others in engagement metrics.
- Retention metrics highlight opportunities for audience growth.

---

## 🚀 Future Improvements
- Build a **Power BI dashboard**
- Add **advanced marketing KPIs**
- Perform **predictive engagement analysis**
